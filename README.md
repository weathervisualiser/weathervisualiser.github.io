# Weather Visualiser
Not your regular old weather app.<br><br>
It is a real-time procedural sky and weather renderer that shows the current weather outside in any city of your choice, almost as if you're looking through a window!<br>
It simulates atmospheric lighting, sun and moon cycles, and particle precipitation using only vanilla Javascript and HTML Canvas. It uses Open-Meteo for the weather data and HTML Canvas to display and animate that data.
